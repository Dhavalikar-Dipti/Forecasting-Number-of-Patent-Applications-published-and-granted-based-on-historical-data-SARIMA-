# Forecasting-Number-of-Patent-Applications-published-and-granted-based-on-historical-data-SARIMA-

This project analyzes historical trends in both granted and published patent
applications, crucial indicators of economic development and innovation.
Accurate forecasting of these patent activities benefits businesses, research
institutions, and policymakers for strategic planning and innovation
landscape assessment. This analysis employs the Seasonal Autoregressive
Integrated Moving Average (SARIMA) model to model weekly data for
both patent grants and publications, generating future projections. The
study also examines the impact of outlier handling on forecasting accuracy
for both datasets.
